## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/gorilcubedev/pomodoro-timer.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd pomodoro-timer
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
